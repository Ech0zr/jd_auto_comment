# jd_AutoComment

## 鸣谢
### 

基于jd_AutoComment稍加改良

## 思路

京东自动评价

由爬虫先行对商品的既有评价进行爬取，在此基础上进行自己的评价

## 使用方法

在终端中执行：

```bash
git clone https://github.com/Ech0zr/jd_auto_comment.git
cd jd_auto_comment
pip install -r requirements.txt
python auto_comment_plus.py
```
